# IoT-Based Smart Cradle for Babies

## Overview

This project was created as a 4th-semester IoT field project.

It is a smart baby cradle system that uses sensors, embedded hardware, and automation to help care for a baby more effectively.

## Project Image

![IoT-Based Smart Cradle for Babies](WhatsApp%20Image%202026-05-04%20at%2012.58.25%20PM.jpeg)

## Key Features

- Detects crying and activates a gentle rocking motion to soothe the baby.
- Monitors wetness and raises an alert when needed.
- Tracks temperature to help maintain a comfortable environment.
- Shows the current status of the cradle on an LCD screen in real time.

## Platform

The project is built around the ESP32 microcontroller, making it suitable for future IoT expansion and remote monitoring.

It combines mechanical movement, sensor-based detection, and smart control to solve a real-life problem in a simple and practical way.

## Learning Outcome

This work also reflected our teamwork, problem-solving, and hands-on learning through circuit building, testing, and debugging.
